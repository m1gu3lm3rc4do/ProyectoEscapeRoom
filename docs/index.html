Escape Room 3D

Un juego de preguntas y respuestas con entorno 3D, creado con Django, Streamlit y pygame

🐍 Python 3.12 🦄 Django 5.2 🎮 pygame + PyOpenGL 📊 Streamlit 🔑 JWT Auth 🗄️ SQLite

¿Qué es Escape Room 3D?

Escape Room 3D es una aplicación de entretenimiento educativo que combina un juego de preguntas y respuestas con un entorno tridimensional renderizado en tiempo real. El jugador controla un personaje animado dentro de una sala 3D y debe responder correctamente un número mínimo de preguntas para "escapar" a través de la puerta.

El proyecto está compuesto por tres capas independientes que se comunican a través de una API REST: un backend Django que gestiona usuarios, sesiones y rooms; un dashboard Streamlit que sirve de interfaz web; y un cliente de juego pygame/OpenGL que renderiza la sala 3D.

Además del modo clásico con preguntas predefinidas, el proyecto incluye un Creator Mode que permite a cualquier usuario registrado diseñar sus propias Escape Rooms con preguntas personalizadas y publicarlas en el catálogo público.

Stack tecnológico

🦄

Django 5.2

Framework web backend. ORM, migraciones, admin y gestión de usuarios.

🔌

Django REST Framework

API REST con ViewSets, serializers y permisos granulares.

🔑

SimpleJWT

Autenticación stateless con access token (8 h) y refresh token (7 días).

📊

Streamlit

Dashboard web interactivo con tabs, gráficos Plotly y gestión de estado.

🎮

pygame

Motor de juego 2D/3D. Gestión de eventos, fuentes, superficies y HUD.

🧊

PyOpenGL

Renderizado 3D con OpenGL legacy (glBegin/glEnd). Sala, personaje y antorchas.

🗄️

SQLite

Base de datos embebida. Sin configuración adicional, ideal para desarrollo.

📦

requests / Plotly

Cliente HTTP para el game client y gráficos interactivos en el dashboard.

Características principales

  • 🏰 Sala 3D con personaje animado — Entorno renderizado con OpenGL. El personaje camina hacia la puerta con cada respuesta correcta y retrocede con cada fallo.
  • 🔥 Antorchas con parpadeo dinámico — Efecto de llama animada usando math.sin(t) con interpolación de color naranja→amarillo a ~8 Hz.
  • ❓ Sistema de preguntas y respuestas — HUD 2D renderizado como textura OpenGL con alpha blending. Teclas 1–4 para responder.
  • ⚒️ Creator Mode — Crea tus propias Escape Rooms con título, descripción, dificultad y hasta 62 preguntas personalizadas.
  • 🗺️ Catálogo público — Explora y juega rooms publicadas por otros usuarios. Filtra por dificultad: Fácil, Medio o Difícil.
  • 📊 Dashboard web con estadísticas — Historial de partidas, tablero de mejores tiempos, gráficos de rendimiento y estadísticas globales.
  • 🏆 Leaderboard — Top 10 de mejores tiempos en el modo clásico, accesible públicamente sin autenticación.
  • 🎨 Interfaz estilo RPG — Paleta oscura con acentos dorados, bordes pergamino, efectos de flash y pantalla de título animada.
